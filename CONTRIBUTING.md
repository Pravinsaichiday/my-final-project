# Contributing Guidelines

Thank you for your interest in contributing to this project! We welcome all contributions.
1.  **Fork** this repository.
2.  Create a **new branch** for your work (`git checkout -b feature/my-new-feature`).
3.  Make your changes and write a clear **commit message** (`git commit -m 'Add some feature'`).
4.  **Push** your branch to your fork (`git push origin feature/my-new-feature`).
5.  Open a **Pull Request** to the main repository.
